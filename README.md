# AI-Powered-Resume-Builder-Backend

This repository contains the backend microservices for the ResumeAI project.

## Project Structure
- `eureka-server`: Service Discovery
- `api-gateway`: API Routing and Security
- `auth-service`: User Identity and Authentication
- `ai-service`: Resume Analysis and ATS Scoring
- `resume-service`: Core Resume Management
- `section-service`: Resume Section Logic
- `notification-service`: Email and Notifications
- `payment-service`: Credit and Transactions
- `job-service`: Job Descriptions and Matching
- `template-service`: Dynamic PDF Generation
